# NodeJs-Treinamento

Curso focado em Node.js, cobrindo desde fundamentos — como modelo assíncrono, event loop e arquitetura single-thread — até a construção de APIs RESTful completas, seguras e escaláveis. Inclui implementação de métodos HTTP, roteamento, manipulação de dados, autenticação com JWT, boas práticas de segurança, proteção contra vulnerabilidades comuns e técnicas de otimização de desempenho.

O conteúdo combina teoria e prática, preparando o aluno para desenvolver e manter aplicações backend prontas para produção. Como complemento, o curso inclui TypeScript aplicado ao ecossistema JavaScript, ampliando a robustez e a qualidade do código.
